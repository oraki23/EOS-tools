# Create Wallet

``` 
cleos wallet create --to-console
```

# Open the wallet

```
cleos wallet open

cleos wallet list
```

# Unlock a wallet

```
cleos wallet unlock
```

# Import keys into a Wallet

```
cleos wallet create_key

cleos wallet import
```

# Create Accounts

- Each account has a "owner" key and "active" key

```
cleos create account eosio bob

cleos get account alice
```
